What is this?


If you're here, you probably know about Gutenberg. But in case you don't, here's the skinny - it's the new, user-friendly visual editor for WordPress. You can install and activate it now, but starting with WordPress 5.0, Gutenberg is slated to be the default editor. Gutenberg allows you to build WordPress pages with blocks - essentially bundles of styled content. Gutenberg comes with default blocks - but Block Party blocks extend Gutenberg to another level.
